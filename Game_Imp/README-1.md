## 🎯 كيفية اللعب

### الإعداد
1. **اللاعب الأول** يدخل اسمه → يضغط "انضم أو أنشئ غرفة" (بدون كود)
2. سيُنشأ **كود غرفة** تلقائياً (مثل: `GAME`)
3. **باقي اللاعبين** يدخلون أسماءهم + كود الغرفة → ينضمون

### الجولة
1. **المضيف** يختار موضوع + عدد الأغراب + وقت النقاش
2. المضيف يضغط **"ابدأ اللعبة"**
3. كل لاعب يضغط على **اسمه** ليرى دوره بسرية
4. بعد كشف الجميع، يبدأ **النقاش** مع مؤقت مزامن
5. **التصويت** — كل لاعب يصوت ضد من يشك فيه
6. **النتيجة** تُكشف للجميع

## 🔧 الأحداث المتزامنة (Socket.io Events)

### من العميل → الخادم
| الحدث | البيانات | الوصف |
|-------|----------|-------|
| `join_room` | `{room, player_name}` | الانضمام لغرفة |
| `start_game` | `{room, theme, impostor_count, timer_seconds}` | بدء اللعبة (المضيف فقط) |
| `reveal_role` | `{room, player_index}` | تأكيد رؤية الدور |
| `start_timer` | `{room, duration}` | بدء المؤقت المزامن |
| `pause_timer` | `{room}` | إيقاف مؤقت |
| `reset_timer` | `{room, duration}` | إعادة المؤقت |
| `chat_message` | `{room, sender, message, emoji_only}` | إرسال رسالة |
| `typing` | `{room, player, is_typing}` | مؤشر الكتابة |
| `cast_vote` | `{room, voter_name, target_index}` | تسجيل تصويت |
| `reveal_result` | `{room}` | كشف النتيجة |
| `go_phase` | `{room, phase}` | تغيير المرحلة |
| `new_round` | `{room}` | جولة جديدة |

### من الخادم → العميل
| الحدث | الوصف |
|-------|-------|
| `joined` | تأكيد الانضمام + بيانات الغرفة |
| `player_joined` | لاعب جديد انضم |
| `player_left` | لاعب غادر |
| `game_started` | بيانات اللعبة (دور اللاعب الخاص — private) |
| `reveal_status_update` | تحديث حالة الكشف |
| `all_revealed` | الجميع كشف دوره |
| `phase_changed` | تغيرت مرحلة اللعبة |
| `timer_started` | بدأ المؤقت |
| `timer_tick` | تحديث المؤقت كل ثانية |
| `timer_paused` | توقف المؤقت |
| `timer_reset` | إعادة المؤقت |
| `timer_ended` | انتهى الوقت |
| `chat_message` | رسالة دردشة جديدة |
| `player_typing` | مؤشر كتابة لاعب |
| `vote_update` | تحديث نتائج التصويت |
| `game_result` | النتيجة النهائية |
| `new_round_started` | بدأت جولة جديدة |
| `error` | رسالة خطأ |

## 🔒 الأمان
- كل لاعب يتلقى **دوره فقط** (private emit) — لا يمكن لأحد رؤية دور الآخرين
- المؤقت مُتزامن من الخادم — لا يمكن التلاعب بالتوقيت
- التصويت مُسجّل في الخادم — لا يمكن التصويت مرتين
- المضيف فقط يملك صلاحية التحكم
